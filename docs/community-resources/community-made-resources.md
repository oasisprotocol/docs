# Community-Made Tools

Below are a number of community-built tools you can use to monitor the Oasis Testnet, automate your node deployment, and more. For more information on building your own integration, check out [the Oasis Core Developer Docs](https://docs.oasis.dev/oasis-core).

## Block Explorers & Validator Leaderboards <a href="block-explorers-validator-leaderboards" id="block-explorers-validator-leaderboards"></a>

* [Oasisscan (by bitcat)](https://oasisscan.com)
* [Oasis Monitor (by Everstake)](https://oasismonitor.com)
* [Oasis Hubble (by Figment Networks)](https://hubble.figment.io/oasis/chains/testnet-beta)
* [Oasis.Fish (by Stakefish)](https://oasis.fish/leaderboard/)
* [Oasis Smartstake (by Smartstake)](https://oasis.smartstake.io)
